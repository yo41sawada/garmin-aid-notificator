# CLAUDE.md

## プロジェクト概要

Garmin Forerunner 165 向けの Connect IQ DataField アプリ。ランニング中に補給タイミングをバイブレーションとトーンで通知し、次の補給ポイントまでの残り距離を表示する。

- **アプリ種別**: DataField（既存のトレーニング画面の1スロットに表示）
- **対応デバイス**: Forerunner 165
- **開発言語**: Monkey C（Connect IQ SDK 9.1.0）

## 開発上の注意

- ソースコードを変更する PR では、**必ず** `README.md` を含む関連ドキュメントおよび PR の概要（description）への影響を確認すること。コミットを重ねた場合も PR の概要が最終的な変更内容と一致しているか都度確認する。更新が必要な場合は同じ PR に含める。「変更なし」と判断した場合もその理由を意識すること。

## コーディングプラクティス

- **アーリーリターン**: 条件を満たさない場合は早期に `return` し、ネストを深くしない
- **関心の分離**: ロジックは `compute()`、描画は `onUpdate()` に分ける
- **単一責任**: メソッドは1つの責務のみ持つ。複数の処理が混在する場合は別メソッドに切り出す
- **定数化**: マジックナンバーは定数として定義する
